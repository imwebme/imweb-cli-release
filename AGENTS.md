# AGENTS.md

## Public Documentation Language Policy

- Keep the root `README.md` in English.
- Keep localized root README files beside it:
  - `README.ko.md` for Korean
  - `README.ja.md` for Japanese
  - `README.zh-CN.md` for Simplified Chinese
- When changing public README content, update all localized README files in the same change.
- Keep the language switcher near the top of every README file.
- Do not add source code, internal operating notes, secrets, private repo references, or unofficial package-manager install instructions.
- Official install instructions are limited to `install/latest.sh` for macOS/Linux and `install/latest.ps1` for Windows.
